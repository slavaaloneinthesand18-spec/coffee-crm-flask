# CRM-система для автоматизации деятельности кофейни «Take a break»

Веб-ориентированная CRM-система на базе Flask для управления продажами, автоматического учета складских остатков и ведения бонусной программы лояльности клиентов.

## Технологический стек
**Backend:** Python 3.10+, Flask, SQLAlchemy
**Database:** SQLite
**Frontend:** HTML5, CSS3, Bootstrap 5, JavaScript

## Функционал
* Авторизация сотрудников с разграничением прав (Администратор / Бариста).
* Защита от перебора паролей (Rate Limiting).
* Интерактивный терминал продаж с корзиной и поиском по номеру телефона.
* Автоматическое списание ингредиентов со склада при создании чека.
* Монитор заказов для вывода клиентам в зале кофейни.

## Запуск проекта
1. Установите зависимости: `pip install flask flask_sqlalchemy werkzeug`
2. Запустите сервер: `python app.py`
3. Откройте в браузере: `http://127.0.0.1:5000`
